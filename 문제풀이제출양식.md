### 주석이 가득 담긴 코드:
```python

#N : 상품 가격, M : 고객이 낸 돈
def change(N,M):
    cnt = 0
    money = [500,100,50,10]
    
    n = M-N # 거슬러줘야 하는 돈
    
    for m in money: #O(m) 동전 종류의 수
        cnt += (n//m) # 나눠주는 횟수
        n %= m
    
    return cnt 

## 총 시간복잡도 : O(M)

```
