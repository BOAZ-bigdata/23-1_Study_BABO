### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    이 과정이 1을 만드는 최소 연산횟수라는 것이 신기하다
    
### 코드의 시간복잡도와 그 이유:
    O(n)
    운이 나빠 k값이 1이라면 최대 n번까지 반복하게 된다
    
    
### 주석이 가득 담긴 코드:
```python
# 이 아래에 코드입력해주세요!!

n, k = map(int, input().split())
result = 0

# N이 K 이상이라면 K로 계속 나누기
while n >= k:
    # N이 K로 나누어 떨어지지 않는다면 N에서 1씩 빼기
    while n % k != 0:
        n -= 1
        result += 1
    # K로 나누기
    n //= k
    result += 1

# 마지막으로 남은 수에 대하여 1씩 빼기
while n > 1:
    n -= 1
    result += 1

print(result)

# 밑에 지우면 안올라가요!
```
