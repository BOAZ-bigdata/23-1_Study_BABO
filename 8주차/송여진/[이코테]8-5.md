### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    함수의 연산 횟수를 출력하므로 다이나믹 프로그래밍을 이용해 반복문을 돌려서 해결한다.
    
### 코드의 시간복잡도와 그 이유:
    다이나믹 프로그래밍 문제이므로 O(N)


### 코드:
```python
x = int(input())

d = [0] * 30001

for i in range(2, x+1):
  d[i] = d[i-1] + 1
  
  if i % 2 == 0:
    d[i] = min(d[i], d[i//2] + 1)
  if i % 3 == 0:
    d[i] = min(d[i], d[i//3] + 1)
  if i % 5 == 0:
    d[i] = min(d[i], d[i//5] + 1)
    
    
print(d[x])
```
