### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    DP테이블 만들어서 저장해주기! , 계단 2개 이하일땐 다 더해버리기!
    
### 코드의 시간복잡도와 그 이유:
    O(N): for문 1개
    
### 주석이 가득 담긴 코드:
```python
# [백준] 2579_계단오르기

N = int(input())

a = [0] * 1001
for i in range(N):
    a[i] = int(input())
    
d = [0] * 1001

if N <= 2: # 계단 2개 이하일땐 다 더해버리기
    print(sum(a))
else:
    d[0] = a[0]
    d[1] = a[0]+a[1]
    for i in range(2,N):
        d[i] = max(d[i-3]+a[i-1] +a[i], d[i-2]+a[i]) # d[i-3]에 앞에거랑 자기자신 더하거나, d[i-2]에서 두칸 띄어서 자기자신 더하거나 그 중에서 max
    print(d[N-1])

```
