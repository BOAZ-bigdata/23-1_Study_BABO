### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    메모이제이션 : 한번 구한 결과를 메모리 공간에 메모해두고 같은 식을 다시 호출해 메모한 결과를 그대로 가져오는 기법
    
### 코드의 시간복잡도와 그 이유:
    O(N) : f(1)을 구한 다음 그 값이 f(2) 푸는데 사용되고, 그게 f(3) 푸는데 사용되고 ,,, 한번 구한 결과는 다시 구해지지 않음!
    
    
### 주석이 가득 담긴 코드:
```python
d = [0] * 100 

def pibo(x):
    print('f(' + str(x) + ')', end = " ")
    if x == 1 or x == 2: 
        return 1
    if d[x] != 0:
        return d[x]
    
    d[x] = pibo(x-1) + pibo(x-2)
    return d[x]

pibo(6)
    
```
