### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    입력을 더 빨리 받는 방법을 알게 되었다.
    
### 코드의 시간복잡도와 그 이유:
    O(1)
    단순 입력과 출력 밖에 없으므로 O(1)이다.
    
    
### 주석이 가득 담긴 코드:
```python
# 이 아래에 코드입력해주세요!!

import sys
# 하나의 문자열 데이터 입력받기
input_data = sys.stdin.readline().rstrip()

# 입력받은 문자열 그대로 출력
print(input_data)

# 밑에 지우면 안올라가요!
```
