### 코드의 시간복잡도와 그 이유:
    O(n)
    반복문이 최대 이다.
    
    
### 주석이 가득 담긴 코드:
```python
# 이 아래에 코드입력해주세요!!

# N(가게의 부품 개수)을 입력받기
n = int(input())
array = [0] * 1000001

# 가게에 있는 전체 부품 번호를 입력받아서 기록
for i in input().split():
    array[int(i)] = 1

# M(손님이 확인 요청한 부품 개수)을 입력받기
m = int(input())
# 손님이 확인 요청한 전체 부품 번호를 공백으로 구분하여 입력
x = list(map(int, input().split()))

# 손님이 확인 요청한 부품 번호를 하나씩 확인
for i in x:
    # 해당 부품이 존재하는지 확인
    if array[i]==1:
        print('yes', end=' ')
    else:
        print('no', end=' ')
```
